# It's `go-time`

A clock for the terminal.

## Commands

### `time`

Shows the current time

```sh
go-time
go-time now
go-time time
```

### `date`

Shows the current date

```sh
go-time date
```

### `cal`

Shows the calendar

```sh
go-time calendar
```
